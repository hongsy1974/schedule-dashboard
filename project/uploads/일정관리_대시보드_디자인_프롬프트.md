# 클로드 디자인용 프롬프트: 일정관리 대시보드

아래 내용을 그대로 복사하여 사용하세요.

---

개인 일정관리 대시보드 웹 애플리케이션을 만들어줘. 아래 요구사항, 사용 환경, 디자인 원칙을 모두 반영해야 해.

## 1. 제품 개요

반복 일정 자동 등록 + 지속 업무 진행률 관리 + 연간 목표 이행 추적을 하나의 대시보드에서 제공하는 개인 일정관리 도구야. 사용자는 연간 반복 업무, 수시 발생 업무, 연간 목표를 동시에 관리하는 사무직 실무 담당자 1인이야.

업무는 3가지 유형으로 구분돼:
- **반복 일정**: 매년/매분기/매월 주기로 자동 생성되는 일정 (예: 연말정산 안내, 분기 보고)
- **지속 업무**: 수시 발생하며 기간을 두고 진행률(%)을 갱신해 완료까지 관리하는 업무
- **목표 과제**: 연초 수립한 목표에 연결된 실행 계획 (지속 업무가 목표에 연결될 수 있음)

## 2. 화면 구성

### 메인 대시보드 (홈)
- **주간 일정 위젯**: 이번 주(월~일) 일정을 요일별로 표시, 오늘 강조, 지연 항목은 경고색 표시, 주 이동 버튼
- **월간 일정 위젯**: 월 캘린더 형태, 일자별 일정 표시, 월 이동 버튼
- **지속 업무 위젯**: 진행 중 업무 카드 목록 — 업무명, 마감일 D-day, 진행률 바(%), 우선순위 뱃지, 7일 이상 미갱신 시 '정체' 표시
- **우선순위 Top 5 위젯**: 중요도×긴급도 점수 상위 업무 정렬 표시
- **목표 달성 현황 위젯**: 연간 목표별 달성률 게이지(도넛 또는 원형 게이지), 시간 경과율 대비 달성률이 낮으면 지연 경고

### 업무 목록/관리 화면
- 표 형태 전체 조회: 업무명, 유형, 시작일, 마감일, 중요도, 긴급도, 우선순위 점수, 진행률, 상태
- 필터(유형/상태/기간), 정렬(우선순위/마감일), 인라인 진행률·상태 변경

### 업무 등록/수정 모달
- 업무명, 설명, 유형, 시작일/마감일, 중요도(상/중/하), 긴급도(상/중/하), 반복 규칙, 연결 목표, 진행률, 메모

### 반복 일정 설정 화면
- 반복 규칙 목록: 주기(매년/매분기/매월), 생성 시점, 사전 알림일(N일 전), 활성/비활성 토글, 다음 생성 예정일 미리보기

### 연간 목표 관리 화면
- 연도별 목표 목록과 목표별 달성률, 목표에 연결된 실행 계획(업무) 목록과 진행률

## 3. 핵심 기능 로직

- 우선순위 점수 = 중요도×2 + 긴급도 + (마감 D-3 이내면 +1). 중요도/긴급도는 상=3, 중=2, 하=1
- 아이젠하워 매트릭스(중요×긴급 4분면) 뷰 제공
- 반복 규칙 등록 시 해당 시기에 업무 자동 생성, 사전 알림일부터 대시보드 상단 알림 영역에 노출
- 목표 달성률 = 연결된 업무 진행률의 평균, 자동 집계
- 업무 추가/수정 시 즉시 대시보드 반영
- 상태: 예정 → 진행중 → 완료, 마감 경과 시 '지연' 자동 표시
- 데이터는 세션 내 메모리 상태로 관리하고, 시연용 샘플 데이터를 미리 채워 넣어줘

## 4. 사용 환경 (중요)

- **담당자는 컴퓨터 모니터(데스크톱)를 주로 사용한다.**
- 기준 해상도 1920×1080, 최소 1440px 폭에 최적화된 데스크톱 우선 레이아웃으로 설계할 것
- 넓은 화면을 활용해 대시보드 위젯을 2~3열 그리드로 배치하고, 스크롤을 최소화할 것
- 마우스/키보드 조작 기준: 호버 효과, 툴팁, 우클릭이 아닌 명시적 버튼 중심 UI
- 모바일 대응은 불필요 (반응형은 고려하지 않아도 됨)

## 5. 디자인 원칙 (사내 포털 스타일 준수)

사내 그룹웨어 포털과 동일한 룩앤필로 만들어줘. 다음 원칙을 지켜:

### 컬러
- **주 색상(Primary)**: 오렌지 계열 `#F37321` (한화 오렌지) — 로고 영역, 활성 탭, 주요 버튼(CTA), 강조 수치, 게이지 차트에 사용
- **배경**: 흰색 `#FFFFFF` 기본, 위젯 구분용 밝은 회색 `#F5F6F7`
- **텍스트**: 진회색 `#333333` 본문, 보조 텍스트 `#888888`
- **경고/지연**: 레드 `#E53935`, 완료: 그린 `#43A047` — 포인트로만 최소 사용
- 전체적으로 화이트 베이스에 오렌지 포인트, 색 사용은 절제할 것

### 레이아웃
- 상단에 수평 GNB(글로벌 내비게이션 바): 메뉴 항목을 가로로 나열하고 활성 메뉴는 오렌지 강조
- 본문은 카드형 위젯 그리드: 각 위젯은 흰색 카드 + 얇은 테두리(또는 미세한 그림자) + 작은 라운드 코너(4~8px)
- 위젯 헤더: 좌측 굵은 제목 + 우측 '+' 또는 '더보기' 아이콘, 헤더 하단 구분선
- 공지/일정 리스트는 좌측 제목·우측 날짜 정렬의 심플한 행 목록 스타일
- 우측 가장자리에 세로 탭(예: WORK / INFO)처럼 보조 패널 접근 탭을 둘 수 있음 (선택)

### 타이포그래피와 컴포넌트
- 한국어 산세리프 폰트 (Noto Sans KR 또는 Malgun Gothic 계열), 본문 13~14px, 위젯 제목 15~16px 볼드
- 진행률·달성률은 도넛/원형 게이지 + 큰 퍼센트 숫자(오렌지)로 표시 (사내 포털의 사용량 게이지 스타일)
- 주요 버튼은 오렌지 배경 + 흰 글자의 알약형(pill) 또는 라운드 사각 버튼
- 뱃지는 작고 각진 라운드 형태: 우선순위(오렌지), 지연(레드), 완료(그린), 유형(회색 아웃라인)
- 장식적 요소, 그라데이션, 과한 애니메이션은 배제하고 실무형·정보 밀도 높은 깔끔한 업무 포털 느낌을 유지할 것

## 6. 완성 기준

- 대시보드 한 화면에서 주간 일정, 월간 일정, 지속 업무 진행률, 우선순위 Top 5, 목표 달성률을 모두 확인 가능
- 업무 등록/수정/삭제, 진행률 변경이 즉시 화면에 반영
- 우선순위 점수 자동 산정 및 정렬 동작
- 반복 규칙 화면에서 다음 생성 예정일 확인 가능
- 샘플 데이터로 모든 위젯이 채워진 상태로 시연 가능할 것
